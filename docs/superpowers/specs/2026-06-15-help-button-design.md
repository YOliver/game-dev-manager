# 帮助按钮功能设计

**日期**：2026-06-15
**状态**：待实现

## 1. 概述

在 Game Dev Manager 应用的菜单栏添加「帮助」菜单，用户点击菜单项后弹出模态对话框，展示 `helpdocs/` 目录中的 Markdown 帮助文档。

## 2. 需求

| 需求 | 说明 |
|------|------|
| 位置 | 菜单栏添加「帮助」菜单 |
| 展示方式 | 模态对话框，内嵌 Markdown 渲染 |
| 菜单项 | 使用手册、欢迎指南、关于（三项都放） |
| 渲染方式 | Markdown 转 HTML 后显示 |
| 对话框特性 | 可调整大小、支持搜索 |
| 搜索功能 | 高亮显示匹配项，支持上一个/下一个跳转 |
| 打包处理 | 将 `helpdocs/` 打包到 exe 内 |

## 3. 架构设计

### 新增/修改文件

| 文件 | 操作 | 说明 |
|------|------|------|
| `gdm/gui/help_dialog.py` | 新增 | 帮助对话框类，负责 MD 转 HTML 和搜索高亮 |
| `gdm/gui/main_window.py` | 修改 | 在菜单栏添加「帮助」菜单 |
| `requirements.txt` | 修改 | 添加 `markdown` 依赖 |

## 4. 帮助对话框设计

### 对话框布局

```
┌─────────────────────────────────────────┐
│  使用手册                    🔍 [搜索框]│
├─────────────────────────────────────────┤
│                                         │
│  （QTextBrowser 显示 HTML 内容）        │
│                                         │
│                                         │
│                                         │
├─────────────────────────────────────────┤
│  ◀ 上一个   下一个 ▶     第 3/10 项   │
└─────────────────────────────────────────┘
```

### 组件说明

| 组件 | 类型 | 功能 |
|------|------|------|
| 标题栏 | - | 显示当前文档标题（如"使用手册"） |
| QTextBrowser | 中央部件 | 显示渲染后的 HTML 内容 |
| 搜索框 | QLineEdit | 输入搜索关键字，实时触发搜索 |
| 上一个/下一个按钮 | QPushButton | 在搜索结果间跳转 |
| 计数器 | QLabel | 显示"第 X/Y 项"，无搜索时隐藏 |

### 关键行为

- **实时搜索**：搜索框文字变化时，自动高亮所有匹配项，并跳转到第一个
- **高亮实现**：通过操作 `QTextDocument` 设置 `QTextCharFormat` 背景色
- **窗口大小**：默认 700×500，可拖动边缘调整

## 5. Markdown 转 HTML 实现

### 依赖安装

```bash
pip install markdown
```

`requirements.txt` 添加：

```
markdown>=3.5.0
```

### 转换逻辑（`help_dialog.py`）

使用 Python `markdown` 库将 Markdown 转换为 HTML，并添加 CSS 样式：

- 支持表格（`tables` 扩展）
- 支持代码块（`fenced_code` 扩展）
- 支持代码高亮（`codehilite` 扩展，可选）
- 支持换行转 `<br>`（`nl2br` 扩展）

### 文档加载

- 开发环境：直接从 `helpdocs/` 目录读取
- 打包环境：使用 `sys._MEIPASS` 访问打包后的资源

## 6. 搜索功能实现

### 实现方案

使用 `QTextDocument` 操作实现搜索高亮：

1. 使用 `QTextDocument.find()` 查找所有匹配项
2. 对每个匹配项设置 `QTextCharFormat` 背景色（黄色高亮）
3. 保存所有高亮的 `QTextCursor` 用于跳转

### 搜索框行为

| 行为 | 实现 |
|------|------|
| 用户输入文字 | 实时触发搜索，高亮所有匹配项 |
| 点击"上一个" | 跳转到上一个匹配项，循环到末尾 |
| 点击"下一个" | 跳转到下一个匹配项，循环到开头 |
| 无匹配项 | 显示"无结果"，禁用跳转按钮 |

## 7. 打包处理

### PyInstaller 配置

修改 `GameDevManager.spec` 文件，将 `helpdocs/` 目录打包到 exe 内：

```python
a = Analysis(
    ['gdm/main.py'],
    datas=[('helpdocs', 'helpdocs')],
    ...
)
```

### 运行时路径访问

在 `help_dialog.py` 中添加 `get_help_doc_path()` 函数，兼容开发环境和打包环境：

- 开发环境：从项目根目录下的 `helpdocs/` 读取
- 打包环境：从 `sys._MEIPASS/helpdocs/` 读取

## 8. 错误处理

| 场景 | 处理方式 |
|------|----------|
| 帮助文档文件不存在 | 弹出 `QMessageBox.warning()` 提示"帮助文档缺失，请联系开发者" |
| Markdown 转换失败 | 捕获异常，降级显示原始 Markdown 文本 |
| 搜索时无匹配项 | 显示"无结果"，禁用上一个/下一个按钮 |
| 打包后路径访问失败 | 捕获 `FileNotFoundError`，提示用户重新安装应用 |
