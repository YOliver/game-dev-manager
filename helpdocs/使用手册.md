# Game Dev Manager 使用手册

## 启动方式

### 方式一：直接运行源码

```bash
pip install -r requirements.txt
python -m gdm.main
```

### 方式二：使用启动脚本

双击 `start.bat`（Windows）

### 方式三：安装包

运行安装包后，从开始菜单或桌面快捷方式启动。

## 打开文件方式

1. **菜单栏**：点击"文件"→"打开文件夹"，选择包含精灵图的目录
2. **拖拽**：将文件夹拖拽到程序窗口（待实现）
3. **命令行**：`python -m gdm.main <文件夹路径>`

## 菜单栏功能说明

| 菜单 | 功能 |
|------|------|
| 文件 → 打开文件夹 | 选择精灵图目录 |
| 文件 → 退出 | 关闭程序 |
| 工具 → 批量重命名 | 打开重命名配置对话框 |
| 帮助 → 使用手册 | 打开本使用手册 |
| 帮助 → 关于 | 显示版本信息 |

## 快捷键

| 快捷键 | 功能 |
|--------|------|
| Ctrl + O | 打开文件夹 |
| Delete | 删除选中项（待实现） |
| F2 | 重命名（待实现） |
| F5 | 刷新当前目录 |

## 自动刷新机制

程序会监控当前打开的目录，当目录中的文件发生变化时（新增、删除、修改），会自动刷新缩略图列表。

## 支持的图片格式

- PNG
- JPG / JPEG
- GIF
- BMP
- 其他 Qt 支持的图片格式

## 批量重命名

支持四种重命名模式：

1. **前缀 + 序号**：如 `sprite_001.png`、`sprite_002.png`
2. **查找替换**：查找指定文本并替换为新文本
3. **正则替换**：使用正则表达式进行高级替换
4. **添加后缀**：在文件名末尾添加指定文本

### 使用步骤

1. 在缩略图视图中选中要重命名的图片（支持多选）
2. 点击"工具"→"批量重命名"
3. 选择重命名模式，配置参数
4. 点击"预览"查看重命名效果
5. 确认无误后点击"执行"

## 典型使用场景

### 场景一：快速预览精灵图

1. 启动程序
2. 打开精灵图目录
3. 在左侧缩略图网格中浏览所有图片
4. 点击某张图片，右侧详情面板显示预览图和元数据

### 场景二：批量规范化命名

1. 选中多张精灵图
2. 打开批量重命名对话框
3. 选择"前缀 + 序号"模式
4. 输入前缀（如 `character_idle_`）
5. 预览并执行

## 日志与调试

### 查看日志

程序运行时的日志输出到控制台。使用 `start.bat` 启动时，控制台窗口会保留，可以查看 print 输出和异常堆栈。

### 常见问题

**问题：缩略图加载失败**
- 原因：图片文件损坏或格式不支持
- 解决：检查图片文件是否正常，尝试用其他图片查看器打开

**问题：程序启动报错 `ModuleNotFoundError`**
- 原因：依赖未安装
- 解决：运行 `pip install -r requirements.txt` 安装依赖

**问题：批量重命名后文件丢失**
- 原因：重命名操作会直接修改文件名
- 解决：重命名前建议备份，或使用版本控制（Git）

## 技术说明

- 缩略图异步加载，避免 UI 卡顿
- 元数据从图片文件实时读取，保证信息准确
- 支持大尺寸图片的缩放显示
