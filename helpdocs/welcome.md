# Game Dev Manager

游戏开发资源管理工具，帮助游戏开发者高效管理精灵图（Sprite）资源。

## 功能特性

- 精灵图资源扫描与元数据提取
- 缩略图预览（支持异步加载）
- 批量重命名（支持多种模式）
- 详细信息展示
- 拖拽支持

## 快速开始

1. 安装依赖：`pip install -r requirements.txt`
2. 启动程序：`python -m gdm.main` 或双击 `start.bat`
3. 使用菜单栏"文件"→"打开文件夹"选择精灵图目录
4. 开始管理你的精灵图资源
